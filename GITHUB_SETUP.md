# Push to GitHub - Instructions

## Step 1: Create Empty Repository on GitHub

1. Go to https://github.com/Ariful-Islam1011
2. Click "New" (top left)
3. Repository name: `phishing-shield`
4. Description: `Friction for Security - Phishing detection system using friction`
5. Choose "Public"
6. Click "Create repository"

## Step 2: Push Local Code to GitHub

Run these commands in terminal:

```bash
cd /Users/md.arifulislam/phishing-shield

# Add remote origin
git remote add origin https://github.com/Ariful-Islam1011/phishing-shield.git

# Rename branch to main (if needed)
git branch -M main

# Push to GitHub
git push -u origin main
```

If prompted for password: Use your GitHub Personal Access Token
(Settings → Developer settings → Personal access tokens)

## Step 3: Verify

Visit: https://github.com/Ariful-Islam1011/phishing-shield

You should see all your code and README!

## Step 4: Add to Hackathon Submission

Use this URL:
```
source_code_url: https://github.com/Ariful-Islam1011/phishing-shield
```

Done! ✅
